## 0. 데이터 스트럭쳐(자료구조) 수업 개요
생활코딩에서 강의하는 데이터 스트럭쳐 수업은 모든 언어에서 공통으로 사용하는 공통적인 개념에 대해 배운다.

## 1. 데이터 스트럭쳐란 무엇인가?
* 현실을 프로그래밍적으로 표현하는 것.
* 거대한 데이터를 효과적으로 관리하는 것.

### 1. TREE(트리)
ex)디렉토리, 회사 계급

### 2. SET(집합)
ex)농구모임, 영어모임

### 3. GRAPH
ex)지도

## 2. 데이터 스트럭쳐가 어려운 이유
* 실무 경험이 없다
* 공감이 되지 안는다
* 이해가 안된다
* 포기 ㅠㅠ
